# 我的博客

基于 [MkDocs](https://www.mkdocs.org/) 和 [Material for MkDocs](https://squidfunk.github.io/mkdocs-material/) 构建。

主题样式参考 [OI-Wiki](https://github.com/OI-wiki/OI-wiki)。

## 本地运行

```bash
# 安装依赖
pip install mkdocs-material

# 启动本地服务器
mkdocs serve

# 构建静态页面
mkdocs build
```

## 目录结构

```
docs/
  index.md      # 首页
  about.md      # 关于页面
  images/       # 图片资源
mkdocs.yml      # 站点配置文件
```
